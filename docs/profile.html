👤 我的信息

切换用户、填写个人信息后，体重与热量等页面会自动使用当前用户的数据。

当前用户

切换后，其他页面（体重记录、热量统计等）将自动显示该用户的数据。

个人信息

在此填写后，体重记录、热量统计等页面会自动读取当前用户的数据。

CSV 格式：首行可为「日期,体重」或「date,weight」，日期 YYYY-MM-DD，体重为数字（kg）

📁 导出 / 导入 CSV

将当前用户数据导出到已记住的 data 目录，或从该目录加载。首次请用「从 data 目录加载」选择 data 文件夹，之后导出会自动写入该目录。
